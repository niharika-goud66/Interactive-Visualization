## Interactive Visualization Dashboard using Power BI

### Project Overview
This project was completed as part of the SkillCraft Technology Data Science Internship. The objective was to create an interactive dashboard using Power BI to analyze HR Employee Attrition data and identify key factors influencing employee turnover.

### Objectives
- Analyze employee attrition trends.
- Identify departments with the highest attrition.
- Examine the impact of age, gender, and job role on attrition.
- Build an interactive dashboard with filters and visualizations.
- Generate meaningful business insights from the data.

### Tools Used
- Power BI Desktop
- CSV Dataset (HR Employee Attrition)
- Data Cleaning and Transformation

### Data Cleaning Process
- Removed duplicate records.
- Checked for missing values.
- Removed blank rows.
- Verified data types.
- Ensured data consistency for analysis.

### Dashboard Features
#### KPI Cards
- Total Employees
- Employees Left
- Average Age

#### Visualizations
- Employee Attrition by Department
- Employee Attrition by Job Role
- Attrition by Gender (Donut Chart)
- Age-wise Attrition Analysis
- Income vs Attrition Analysis
- Years at Company Trend Analysis

#### Interactive Filters
- Department
- Gender
- Job Role
- Age Group
- Education Level

### Key Insights
- Sales Department recorded the highest employee attrition.
- Employees aged 26–35 showed higher attrition rates.
- Laboratory Technicians and Sales Executives experienced higher turnover.
- Lower monthly income was associated with increased attrition.
- Employee retention varied across departments and job roles.
